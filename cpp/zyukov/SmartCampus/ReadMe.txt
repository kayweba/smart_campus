Для построения проекта необходимо установить Conan, Qt версии 5, а также Cmake не ниже версии 3.15.
Для запуска сборки и компиляции проекта необходимо запустить MakeMe.cmd.
MakeMe.cmd принимает такие параметры как:
1) x64 - для сборки под архитектуру x86_64
2) x32,x86 - для сборки под архитектуру x86
3) release - для сборки релизной версии программы
4) debug - для сборки программы с отладочными символами
После построения, файл программы, библиотеки, а также файлы базы данных и настройки генератора будут находиться по пути ./build/bin/{BUILD_TYPE}
